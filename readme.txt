## Zusätzliche Dateien finden sie in unserer API unter dem Ordner Parts4Games/zusätzlicheDateien.
Er enthält:

Dokumentation_der_Fremdienste.pdf -- Hier stehen Informationen über die API's, die wir für unser Projekt benutzen.
RequestsToImport-Team7.json -- Die ist eine Datei mit den wichtigsten Anfragen für unseren UseCase. Sie kann in gängigen Programmen wie Postman bzw. RESTer importiert werden.
Parts4Games.yaml -- yaml Datei mit der Definition unserer Schnittstelle.
Parts4Games.html -- html Datei mit der Definition unserer Schnittstelle.
BackEnd_ReadMe.txt -- Enthält Informationen mit möglichen Fehlern und Behandlungen.
